MLOps pipeline for real estate investment ROI prediction

## Stack

- Cloud: Yandex Cloud (Terraform)
- Orchestration: Prefect
- Tracking: MLflow
- Inference: FastAPI + vLLM (GPU)
- Monitoring: Prometheus + Grafana
